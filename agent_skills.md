# Software Development Agents

This document outlines the roles and skills of the various agents required for the development project.

## 1. Product Owner and Architect
**Role:** Defines the product vision, gathers requirements, and designs the high-level system architecture.
**Skills:**
- Requirements gathering and analysis
- System architecture design (Microservices, Monolithic, Serverless)
- API design and documentation (OpenAPI/Swagger)
- Agile methodology and backlog management
- Technology stack selection

## 2. Frontend Developer
**Role:** Develops the user interface and client-side logic of the application.
**Skills:**
- HTML5, CSS3, JavaScript
- TypeScript
- Next.js and React
- TailwindCSS for styling
- Node.js (for frontend tooling and SSR)
- Responsive web design and accessibility (a11y)

## 3. Backend Developer
**Role:** Develops the server-side logic, APIs, and business logic.
**Skills:**
- Node.js
- Frameworks like Express.js or NestJS
- RESTful and GraphQL API development
- Authentication and Authorization (JWT, OAuth)
- Performance optimization and caching

## 4. Database Specialist
**Role:** Designs, manages, and optimizes the database schema and queries.
**Skills:**
- PostgreSQL
- Prisma ORM
- Database schema design and normalization
- Query optimization and indexing
- Data migration and backup strategies

## 5. Code Standards Checker
**Role:** Ensures the codebase adheres to predefined coding standards and best practices.
**Skills:**
- Linter configuration and enforcement (ESLint, Prettier)
- Static code analysis (e.g., SonarQube)
- Code review best practices
- Ensuring type safety (TypeScript strict mode)

## 6. Quality Assurance and Testing
**Role:** Ensures the quality of the software through automated and manual testing.
**Skills:**
- Unit testing (Jest, Vitest)
- Integration testing
- End-to-end (E2E) testing (Cypress, Playwright)
- API testing (Postman, Supertest)
- Test-driven development (TDD)

## 7. DevOps Engineer
**Role:** Manages the deployment, scaling, and monitoring of the application.
**Skills:**
- Continuous Integration and Continuous Deployment (CI/CD) pipelines (GitHub Actions, GitLab CI)
- Containerization (Docker)
- Container orchestration (Kubernetes)
- Cloud platforms (AWS, GCP, Azure)
- Infrastructure as Code (Terraform)
- Monitoring and alerting (Prometheus, Grafana)
